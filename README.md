## Config and Install

If you want to use this tool, you just need to clone it, and checkout the gh-pages branch

use:

```
# cp gatherUrl.sh /usr/local/bin/gatherUrl
# gatherUrl -s

```

Now, you can use the `gatherUrl -h` to find how to use it.
